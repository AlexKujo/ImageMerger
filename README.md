## ğŸ¯ How to Use

### Perfect for Corel XVL Workflow

This utility is specifically designed to work with images exported from Corel XVL editor, helping you quickly merge different layers and create professional results.

1. **Load Images from Corel XVL:**
   - **Color Image** - main photo/image for processing
   - **Outline** - black and white contour image (required)
   - **Highlight** - image with red areas for highlighting (optional)

2. **Processing:**
   - The application automatically merges the images
   - Red areas from highlight will be applied to the color image
   - Black areas from outline will darken corresponding parts
   - Background transparency is reduced to highlight objects

3. **Saving:**
   - Click "Save" button to save the result
   - Choose format (PNG or JPEG)
